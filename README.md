​Grand Divina - Landing Page Exclusiva

​Este repositório contém o código-fonte da landing page oficial do Grand Divina Eventos. O projeto foi desenvolvido com foco em exclusividade, alta performance visual e experiência do usuário em dispositivos móveis.

​Aviso de Uso

​Este é um projeto proprietário. O código, as imagens e os vídeos contidos neste repositório são de uso exclusivo do Grand Divina.
​Não é permitida a reprodução, distribuição ou uso comercial deste código sem autorização prévia.

​O acesso ao código é disponibilizado aqui apenas para fins de backup e demonstração de portfólio técnico.

​Destaques Técnicos (Para Desenvolvedores)

​Para quem está explorando o código por curiosidade, aqui estão as soluções implementadas:

​Custom UI/UX: Interface construída do zero (Vanilla CSS/JS) para garantir leveza e zero dependência de frameworks pesados.

​Gestão de Histórico: Implementação de History API para interceptar o botão "voltar" do Android/iOS, melhorando a retenção no site.
​Performance de Mídia: Carregamento dinâmico de elementos <video> e <img> via JavaScript para otimizar o tempo de carregamento inicial (LCP). Mobile First Core: Design pensado para navegação com uma mão, deslocando elementos interativos para a "zona de alcance" na base da tela.

​Organização
​index.html: Estrutura semântica otimizada para SEO.
​style.css: Estilização modular com foco em responsividade extrema.
​script.js: Motor de interação da galeria e controle de estado do modal.
